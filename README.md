## Build Setup
Using npm:
```bash
$ npm install
$ npm install vuex-class-modules
```
Using npm axios:
```bash
$ npm install axios
```
Using npm Install Css:
```bash
$ npm install --save-dev @nuxtjs/tailwindcss postcss@latest
```
serve with hot reload at localhost:3000
```bash
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
